# Finanzas Proyecto
Sistema para lógica financiera de empresas y startups.

### Comando para correr el programa:
Desde la carpeta raíz:
```bash
python main.py
```

### Comandos instalar librerías necesarias:

1. Crear entorno virtual:
```bash
python -m venv venv
```

2. Activar el entorno virtual:

- En Windows
```bash
venv\Scripts\activate
```

- En Linux/Mac
```bash
source venv/bin/activate
```

3. Instalar las librerías:

```bash
pip install -r requirements.txt
```

#### Para pegar directamente (Windows):
```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```
